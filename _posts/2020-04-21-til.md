**Gatsby CLI 명령어**  

1. Gatsby CLI 설치

    ```
    sudo npm install -g gatsby-cli
    ```

2. Gatsby 프로젝트 생성  

    ```
    gatsby new '프로젝트 이름'
    ```

3. Local build  

    ```
    gatsby develop
    ```

4. Production build

    ```
    gatsby build
    gatsby serve
    ```
